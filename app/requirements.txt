# Gunicorn — Production WSGI Server
# =============================================================================
#
# NEDEN GUNICORN?
# ---------------
# Flask'ın built-in development server'ı (flask run) production'da kullanılmamalıdır:
# 1. Single-threaded: Aynı anda sadece bir istek işleyebilir
# 2. Güvenlik: Güvenlik denetimleri yoktur
# 3. Performans: Yüksek trafik altında çöker
# 4. Flask'ın kendi dokümantasyonu bile bunu uyarır!
#
# Gunicorn ise:
# 1. Multi-worker: Birden fazla istek paralel işlenir
# 2. Pre-fork modeli: Her worker ayrı bir process olarak çalışır
# 3. Graceful restart: Zero-downtime deployment mümkün
# 4. Production-tested: Instagram, Pinterest gibi devler kullanır
gunicorn==23.0.0
